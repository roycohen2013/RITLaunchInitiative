	KS8895/8875/8864 MDC/MDIO SMI Interface Software Tool

======================================================================

Your PC need to have FTDI USB driver installed first, if you installed any switch/mdio configuration before

The FTDI USB drier has been installed already.

Run the ‘MicrelSMIIfApp.exe’ and using ‘>r [register]’ to read and ‘> w [register] [value]’ to write
